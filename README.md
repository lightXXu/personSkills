# Skills

Personal Codex skills.

## Layout

```text
skills/
  <skill-name>/
    SKILL.md
    agents/openai.yaml
    references/
    scripts/
docs/
scripts/
```

## Included Skills

- `llm-wiki-manager`: Manage a SiYuan-based LLM Wiki knowledge system.

## Notes

This repository intentionally keeps skills self-contained. Each skill should carry only the instructions, references, and scripts needed by an agent to perform the work.
